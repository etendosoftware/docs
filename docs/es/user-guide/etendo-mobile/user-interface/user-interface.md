# Interfaz de Usuario

## Etendo Mobile

### Pantalla de configuración

En la ventana de la pantalla de configuración, el usuario puede establecer el **idioma** que se utilizará en la aplicación.

!!!note
    Recuerde que, en cuanto se descarga la aplicación, se debe configurar la URL del servidor. 

![alt text](../../../assets/user-guide/etendo-mobile/user-interface/user-interface-0.png)


### Pantalla de inicio

Después de iniciar sesión, se le redirigirá a la pantalla de inicio.

![alt text](../../../assets/user-guide/etendo-mobile/user-interface/user-interface-1.png)

Aquí, al seleccionar la imagen de perfil, es posible navegar a las pantallas de **Configuración** y **Perfil**.

![alt text](../../../assets/user-guide/etendo-mobile/user-interface/user-interface-2.png)


Si se muestra el panel lateral, es posible navegar a la pantalla de inicio y también a las subaplicaciones cargadas previamente. 

!!! note
    Para poder incluir subaplicaciones, debe estar instalado el bundle Mobile Extensions Bundle. Para ello, siga las instrucciones del marketplace: [Mobile Extensions Bundle](https://marketplace.etendo.cloud/#/product-details?module=55A7EF64F7FA43449B249DA7F8E14589){target="\_blank"}. <br>
    Para más información sobre las versiones disponibles, la compatibilidad con el core y las nuevas funcionalidades, visite [Mobile Extensions Bundle - Notas de la versión](../../../whats-new/release-notes/etendo-mobile/bundles/mobile-extensions/release-notes.md).

![alt text](../../../assets/user-guide/etendo-mobile/user-interface/user-interface-3.png)

### Pantalla de perfil

En esta sección, el usuario puede ver la información de la persona que ha iniciado sesión: **Nombre, Rol, Organización, Cliente y Almacén**.

En caso de que el usuario tenga una imagen, se mostrará tanto en esta ventana como en la sección superior del menú de navegación en Etendo. 

![alt text](../../../assets/user-guide/etendo-mobile/user-interface/user-interface-4.png)

!!!note
    El rol solo puede modificarse desde Etendo Classic, por lo que cada usuario accederá con su rol por defecto configurado.
