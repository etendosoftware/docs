## Módulo

### Módulo de inclusión / exclusión de Etendo BI

:octicons-package-16: Javapackage: `com.etendoerp.powerbi.inclusion.exclusion`

<iframe width="560" height="315" src="https://www.youtube.com/embed/cEuwiUQbUAQ" title="Reproductor de vídeo de YouTube" frameborder="0" allow="accelerometer; autoplay; clipboard-write; encrypted-media; gyroscope; picture-in-picture; web-share" allowfullscreen></iframe>

Este módulo permite a los usuarios personalizar y configurar la información y los filtros en los informes de Power BI. Permite la creación de diferentes configuraciones que facilitan la selección y la exclusión de entidades específicas dentro de los informes.

!!! info
    Para más información, visite la [Guía de usuario de configuración de inclusión/exclusión de Etendo BI](../../../../../user-guide/etendo-classic/optional-features/bundles/etendobi-extensions/inclusion-exclusion-configuration.md).

### Módulo conector de Etendo BI

:octicons-package-16: Javapackage: `com.etendoerp.integration.powerbi`

<iframe width="560" height="315" src="https://www.youtube.com/embed/z9EBff_qBDE" title="Reproductor de vídeo de YouTube" frameborder="0" allow="accelerometer; autoplay; clipboard-write; encrypted-media; gyroscope; picture-in-picture; web-share" allowfullscreen></iframe>

Este módulo permite al usuario crear consultas base y personalizarlas cuando sea necesario. Este proceso permite proporcionar información que posteriormente puede ser utilizada por Power BI. 

Se debe configurar un proceso en segundo plano que cree archivos CSV que se cargarán en un servidor.  

!!! info
    Para más información, visite la [Guía del desarrollador del Módulo conector de Etendo BI](../../../../../developer-guide/etendo-classic/bundles/etendo-bi-extensions.md#etendo-bi-connector).
