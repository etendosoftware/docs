# Procesado del dudoso cobro

:material-menu: `Aplicación` > `Gestión Financiera` > `Cobros y Pagos` > `Transacciones` > `Procesado del dudoso cobro`

La ventana **Procesado del dudoso cobro** se utiliza para definir qué deudas, y en qué porcentaje, se consideran de dudoso cobro. Estos dudosos cobros se muestran posteriormente en la ventana Dudoso cobro.

![Ventana Procesado del dudoso cobro](../../../../../../assets/user-guide/etendo-classic/basic-features/financial-management/receivables-and-payables/transactions/doubtful-debt-run/doubtful-debt-run-1.png)

Algunos campos a destacar:

- **Método de dudoso cobro:** Si existe un Método de dudoso cobro definido, se puede seleccionar para completar automáticamente los campos necesarios con los valores proporcionados.
- **Días de Vencimiento:** Este campo se utiliza como filtro al seleccionar las deudas existentes. Puede eliminarse posteriormente.
- **Porcentaje:** Este campo se utiliza al seleccionar las deudas existentes como el porcentaje predeterminado de la deuda que se considerará de dudoso cobro. Puede modificarse posteriormente.

**Seleccionar Dudoso cobro**. Tras completar los campos necesarios, se muestra el botón "Seleccionar Dudoso cobro". Al hacer clic en él, aparece un nuevo pop-up para seleccionar las deudas que se van a considerar de dudoso cobro.

![Seleccionar Dudoso cobro](../../../../../../assets/user-guide/etendo-classic/basic-features/financial-management/receivables-and-payables/transactions/doubtful-debt-run/doubtful-debt-run-2.png)

En esta ventana se muestran todas las deudas pendientes, prefiltradas por los parámetros establecidos anteriormente. Estos filtros se pueden eliminar haciendo clic en el icono de embudo.

Cada fila representa una Deuda Pendiente, y es posible seleccionar tantas como sea necesario para definirlas como de dudoso cobro.

Para cada línea, es posible editar el campo **Importe de Dudoso cobro**. Este es el importe que se considerará de dudoso cobro. Tenga en cuenta que no es posible definir un Importe de Dudoso cobro mayor que el Importe Pendiente.

El campo **Fecha de Ejecución** también puede modificarse por cada fila. Al hacer clic en el botón Aceptar, los registros seleccionados serán visibles en la pestaña Dudoso cobro que aparece a continuación.

**Procesar**:

Es necesario hacer clic en el botón Procesar después de seleccionar los registros para generar el/los Dudoso/s cobro/s correspondiente/s.

Una vez creados, será posible contabilizarlos o reactivarlos a través de la ventana Dudoso cobro.

## Dudoso cobro

En esta pestaña se muestran los registros seleccionados previamente. A continuación, será necesario Procesar el Dudoso cobro para que esté disponible en la ventana Dudoso cobro.
