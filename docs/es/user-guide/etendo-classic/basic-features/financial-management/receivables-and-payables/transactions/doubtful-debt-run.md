# Ejecución de Deudas Dudosas

:material-menu: `Aplicación` > `Gestión Financiera` > `Cobros y Pagos` > `Transacciones` > `Ejecución de Deudas Dudosas`

La ventana **Ejecución de Deudas Dudosas** se utiliza para definir qué deudas, y en qué porcentaje, se consideran dudosas. Estas deudas dudosas se muestran posteriormente en la ventana Deuda Dudosa.

![Ventana Ejecución de Deudas Dudosas](../../../../../assets/drive/1O6LI7av2TJyYTZyW8aE3gjlakxldIxhy.png)

Algunos campos a destacar:

- **Método de Deuda Dudosa:** Si existe un Método de Deuda Dudosa definido, se puede seleccionar para completar automáticamente los campos necesarios con los valores proporcionados.
- **Días de Vencimiento:** Este campo se utiliza como filtro al seleccionar las deudas existentes. Puede eliminarse posteriormente.
- **Porcentaje:** Este campo se utiliza al seleccionar las deudas existentes como el porcentaje predeterminado de la deuda que se considerará dudosa. Puede modificarse posteriormente.

**Seleccionar Deuda Dudosa**. Tras completar los campos necesarios, se muestra el botón "Seleccionar Deuda Dudosa". Al hacer clic en él, aparece un nuevo pop-up para seleccionar las deudas que se van a considerar dudosas.

![Seleccionar Deuda Dudosa](../../../../../assets/drive/14aXYdvA41Jd2AkO4x7yD3wesdB3q0LBG.png)

En esta ventana se muestran todas las deudas pendientes, prefiltradas por los parámetros establecidos anteriormente. Estos filtros se pueden eliminar haciendo clic en el icono de embudo.

Cada fila representa una Deuda Pendiente, y es posible seleccionar tantas como sea necesario para definirlas como dudosas.

Para cada línea, es posible editar el campo **Importe de Deuda Dudosa**. Este es el importe que se considerará dudoso. Tenga en cuenta que no es posible definir un importe de Deuda Dudosa mayor que el Importe Pendiente.

El campo **Fecha de Ejecución** también puede modificarse por cada fila. Al hacer clic en el botón Aceptar, los registros seleccionados serán visibles en la pestaña Deuda Dudosa que aparece a continuación.

**Procesar**:

Es necesario hacer clic en el botón Procesar después de seleccionar los registros para generar la/s Deuda/s Dudosa/s correspondiente/s.

Una vez creadas, será posible contabilizarlas o reactivarlas a través de la ventana Deuda Dudosa.

## Deuda Dudosa

En esta pestaña se muestran los registros seleccionados previamente. A continuación, será necesario Procesar la Deuda Dudosa para que esté disponible en la ventana Deuda Dudosa.
