# Actividad (ABC)

:material-menu: `Aplicación` > `Gestión Financiera` > `Contabilidad` > `Configuración` > `Actividad (ABC)`

## Visión general

El cálculo de costes basado en actividades (ABC) es un modelo especial de cálculo de costes que identifica actividades en una organización para las cuales se requiere la gestión del coste.

"*Actividad*" es una de las dimensiones que se pueden configurar para una determinada configuración del libro mayor.

### Proceso

La ventana Actividad (ABC) permite al usuario crear tantas actividades como sean necesarias por cada organización.

![](../../../../../../assets/drive/1ekUpsYTliIZF7K_3I_HrqIgD5DGf7vrw.png)

Tal y como se muestra en la pantalla anterior, es posible definir actividades de resumen que soportan una estructura jerárquica o actividades dentro de una organización.
