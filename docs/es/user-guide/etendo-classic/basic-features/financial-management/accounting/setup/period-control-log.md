# Histórico de Control de Periodos

:material-menu: `Aplicación` > `Gestión Financiera` > `Contabilidad` > `Configuración` > `Histórico de Control de Periodos`

## Visión general

Esta ventana muestra el histórico del control de periodos.

Los periodos de un año pueden abrirse, cerrarse y cerrarse permanentemente. La ventana **Histórico de Control de Periodos** muestra todas las aperturas y cierres de los periodos que han tenido lugar en la aplicación; por lo tanto, puede utilizarse para comprender cómo ha cambiado el estado de un periodo a lo largo del tiempo.

Cuando un periodo se abre o se cierra mediante la ventana de control de periodos Abrir/Cerrar, o se ha cerrado permanentemente como resultado del cierre de un año mediante la ventana de cierre de fin de año, se crea un registro en el Histórico de Control de Periodos.

Por ejemplo, esta captura de pantalla muestra el Histórico de Control de Periodos para una organización que ha abierto todos sus periodos para el año 2019:

![](../../../../../../assets/drive/1ZQv4ciDH8NuE6aOQLnWmztUUVg2PxZuN.png)

Tal y como se muestra en la imagen anterior, los campos principales en esta ventana son:

-   Organización.
-   Calendario.
-   Año.
-   Periodo. El periodo inicial a partir del cual se va a realizar la acción de periodo (normalmente coincide con el periodo hasta)
-   Periodo hasta. El periodo final hasta el cual se va a realizar la acción de periodo (normalmente coincide con el periodo)
-   Acción de periodo. La acción que ha tenido lugar. Puede ser Abrir periodo, Cerrar periodo o Cerrar periodo permanentemente.
-   Categoría de documento. Si la acción de periodo se ha realizado solo para un tipo de documento, se mostrará aquí. Esto significa que la acción de periodo ha afectado únicamente al tipo de documento mostrado para ese periodo, no a todos los demás tipos de documento, que permanecen en el mismo estado que antes.
