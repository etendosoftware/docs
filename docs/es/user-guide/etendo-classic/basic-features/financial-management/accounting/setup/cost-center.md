# Centro de costos

:material-menu: `Aplicación` > `Gestión Financiera` > `Contabilidad` > `Configuración` > `Centro de costos`

## Visión general

El Centro de costos es una dimensión contable que puede utilizarse al contabilizar documentos en el libro mayor.

![](../../../../../../assets/drive/1vk6c1L49EaXAtzd0A121Tq55wJZfq1Fu.png)

### Centro de costos

La ventana Centro de costos permite al usuario crear los centros de costos de la organización. Se utiliza como maestro de los centros de costos de una organización.

Un centro de costos puede crearse simplemente introduciendo el nombre y la descripción del centro de costos de la organización. Si se crea un centro de costos para la organización (\*), ese centro de costos estará disponible para todas las organizaciones del cliente. Un centro de costos puede crearse como resumen, lo que ayuda a obtener una estructura en árbol de centros de costos.
