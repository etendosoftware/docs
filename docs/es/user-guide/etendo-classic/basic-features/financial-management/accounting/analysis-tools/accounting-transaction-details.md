# Datos de contabilidad

:material-menu: `Aplicación` > `Gestión Financiera` > `Contabilidad` > `Herramientas de análisis` > `Datos de contabilidad`

## Descripción general

La ventana de datos de contabilidad es una lista detallada de cada asiento contable de un libro mayor.

Etendo dispone de un sistema contable integrado que combina contabilidad financiera y analítica.

-   La **contabilidad financiera** permite al usuario explotar dimensiones contables como la "Organización", la "Cuenta" y la "Fecha contable":
    -   Estas dimensiones son siempre **obligatorias**, lo que significa que deben especificarse cada vez que se contabiliza un documento en el libro mayor.
-   La **contabilidad analítica** permite al usuario explotar otras dimensiones como "Producto", "Tercero" y "Región de ventas".
    -   Estas dimensiones pueden configurarse como obligatorias u opcionales en la configuración del libro mayor de la organización, siempre que la entidad a la que pertenece la organización no **"gestione centralmente"** las dimensiones contables.
    -   De lo contrario, si la entidad **"gestiona centralmente"** las dimensiones contables, algunas dimensiones analíticas pueden configurarse en la ventana Entidad (p. ej., "Producto", "Proyecto", "Centro de costos"), mientras que otras deben configurarse en la configuración del libro mayor de la organización (p. ej., "Región de ventas", "Campaña").

Etendo permite al usuario contabilizar transacciones en el libro mayor solo si se especifican las dimensiones financieras y las dimensiones analíticas obligatorias, mientras que siempre existe la opción de especificar las opcionales.

## Cabecera

Este informe lista cada transacción contabilizada en el libro mayor mostrando cada dimensión contable especificada.

![](../../../../../../../assets/user-guide/etendo-classic/basic-features/financial-management/accounting/analysis-tools/accounting-transaction-details/accounting-transaction-details-1.png)

Los filtros de columna permiten al usuario filtrar la información a mostrar por cualquiera de las dimensiones contables.
