# Albaranes pendientes de recibir

:material-menu: `Aplicación` > `Gestión de Compras` > `Transacciones` > `Albaranes pendientes de recibir`

Esta ventana permite al usuario:

- **Use the Filter options** para acotar la búsqueda de pedidos de compra pendientes de entregar. Es posible buscar por:
    - Tercero
    - Desde Fecha de pedido de compra -> Hasta Fecha de pedido de compra
    - Número de pedido de compra
- Introducir una **"Reception date".**
- **Seleccionar** las **"Order line/s of a purchase order/s" delivered** que se muestran agrupadas por Tercero y Pedido de compra.
- **Cambiar estado** la "**Cantidad**" de los bienes que se están recibiendo si es necesario.
- **Introducir** la "**Goods receipt location**" o ubicación (bin) dentro de un almacén.
- **Process it** para crear los albaranes correspondientes.
