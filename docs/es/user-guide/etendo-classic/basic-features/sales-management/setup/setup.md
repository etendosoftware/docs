## Estado del producto

:material-menu: `Aplicación` > `Gestión de Ventas` > `Configuración` > `Estado del producto`

En esta ventana, el usuario define en qué estado proviene el producto del cliente. Estos valores se utilizan en la ventana **Recibo devolución de material**.

![Estado del producto](../../../../../assets/user-guide/etendo-classic/basic-features/sales-management/setup/condition-of-the-goods-1.png)

## Zona de venta

:material-menu: `Aplicación` > `Gestión de Ventas` > `Configuración` > `Zona de venta`

En esta sección, el usuario puede crear zonas de venta para utilizarlas en las operaciones de ventas y definir una zona de venta que se utilizará en el proceso de ventas.

![Zona de venta](../../../../../assets/user-guide/etendo-classic/basic-features/sales-management/setup/sales-region-1.png)

## Comisión

:material-menu: `Aplicación` > `Gestión de Ventas` > `Configuración` > `Comisión`

En esta ventana, los usuarios pueden configurar planes de comisiones para los representantes de ventas. La funcionalidad de Comisión permite definir cómo y cuándo se calculan las comisiones, y a quién se pagan. Las comisiones pueden calcularse en función del importe neto de ventas o de los márgenes de beneficio, con soporte para escenarios complejos mediante filtros por tercero, producto o zona.

Para obtener información detallada sobre la configuración de comisiones, los métodos de cálculo y ejemplos, consulte la sección [Comisión](./commission.md).

![Ventana de Comisión](../../../../../assets/user-guide/etendo-classic/basic-features/sales-management/setup/commission-1.png)
