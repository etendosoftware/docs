# Gestión del módulo de Empresa

:material-menu: `Aplicación` > `Configuración General` > `Organización` > `Gestión del módulo de Empresa`

Los conjuntos de datos permiten al usuario definir datos de referencia, como datos maestros o datos de configuración, para módulos o incluso para el núcleo de Etendo.

Los datos de referencia se pueden importar o aplicar a organizaciones en la ventana **Gestión del módulo de Empresa** simplemente seleccionando la organización para la cual se deben aplicar los datos de referencia y los datos de referencia que se van a aplicar o a importar.

![](../../../../../assets/drive/GCwDdNOwwnsS4qqc5AOasqxELrfP6UKdw6AY3ffwJuGxF0qL36O0MLh6_BZI9TPXNT8ywlVdzqL7nn4gf-opz7Zxo-tX5Ap1o99nZMhU3fLcn3mofw12qESDtmyyLrJJcRh2zdX7.png)

- Si unos datos de referencia se aplican en la organización (\*), todas las organizaciones del sistema tendrán acceso a esos datos.
- Si unos datos de referencia se aplican en una organización padre, todas las organizaciones que pertenecen a ella tendrán acceso a esos datos.
- Si unos datos de referencia solo se aplican a una organización que no tiene ninguna organización por debajo en el árbol, solo esa organización tendrá acceso a esos datos.
