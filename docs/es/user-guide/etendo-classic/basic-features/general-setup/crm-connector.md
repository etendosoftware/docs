# Configuración del Conector CRM

:material-menu: `Aplicación` > `Configuración General` > `Conector CRM` > `Configuración del Conector CRM`

## Visión general

El conector CRM concreto debe añadir una configuración en esta ventana para permitirle conectarse al sistema externo y para definir las propiedades y filtros utilizados por Etendo para mapear con la información del sistema externo.

![](../../../../assets/user-guide/etendo-classic/basic-features/general-setup/crm-connector/crm-connector.png)

Campos a tener en cuenta:

- **Nombre**: Identificador del conector.
- **Descripción**: Información adicional sobre el conector. 
- **URL**: La URL define una dirección en línea para este elemento.
- **Nombre usuario**: Nombre que identifica al usuario en el sistema.
- **Contraseña**: Indica la contraseña para este ID de usuario. Las contraseñas son necesarias para identificar a los usuarios autorizados.
- **Tiempo de Expiración**: El tiempo de expiración es el valor en segundos establecido por el usuario para la solicitud al servicio definido.

## Propiedad API

Una propiedad API es como un mapa que relaciona una clave de API externa con un mensaje de Etendo. Define el tipo de dato y cualquier otra configuración relacionada.

### Opciones de Propiedad

Permite al usuario definir las opciones aceptadas por la propiedad padre.

## Filtro

Filtros para la integración de terceros. El filtro tiene un tipo definido y está vinculado a una o varias propiedades.

### Opciones del Filtro

Permite al usuario definir las opciones aceptadas por el filtro padre.

### Mapeo de direcciones

En esta solapa, el usuario puede definir direcciones y determinar las propiedades que deben utilizarse para el mapeo de cada columna de ubicación.
