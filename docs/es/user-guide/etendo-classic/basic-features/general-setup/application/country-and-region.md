# País, provincia y ciudad
:material-menu: `Aplicación` > `Configuración General` > `Aplicación` > `País, provincia y ciudad`

## Visión general

Los países y las provincias son configuraciones básicas para compartir dentro de Etendo.

Los países y sus correspondientes provincias se crean y se listan automáticamente en esta ventana una vez que se completa la instalación de Etendo.

!!! info
    Todos estos países y provincias están vinculados a la organización del sistema (\*), lo que significa que esos países y provincias serán compartidos por todas las organizaciones del sistema.

## País, provincia y ciudad

Esta ventana permite al usuario visualizar y/o crear y mantener los países y las provincias que se utilizarán en las actividades empresariales diarias.

![alt text](../../../../../assets/user-guide/etendo-classic/basic-features/general-setup/application/country.png)

![](../../../../../assets/drive/7wlTKkSY89CblxbUfLmnLafkVTgPVlmk7_I-SS8d0mFe3lrRH3_5VN_PvJeClUPP1ZC9uhTTcLDdH3GhKAH2nuqpJgJgAEBwdJnVkTX1GtYu7OmXavxslpmGx84aoeNbNS2m8sCu.png)

Como se muestra en la imagen anterior, los datos relevantes del país son:

- El **código ISO del país**.
    - Los códigos ISO del país son códigos utilizados a nivel mundial para la representación de los países.
- El **Formato impresión dirección**.
- Si tiene algún tipo de **Provincia** como estados o no. El indicador **Tiene región** debe estar habilitado para introducir provincias en la solapa Provincia.
- El **Provincia**.
- El **Código IBAN** del país.
    - IBAN o **Número de Cuenta Bancaria Internacional** es un sistema de numeración que se creó para categorizar las cuentas bancarias a nivel mundial.
- La **longitud del Código IBAN** del país.
- El **formato del número de teléfono**.
- El **idioma** por defecto.
- y la **moneda** por defecto.

### Traducción

Los países y las provincias se pueden traducir a cualquier idioma.

![](../../../../../assets/user-guide/etendo-classic/basic-features/general-setup/application/country-translation.png)

### Provincia

![alt text](../../../../../assets/user-guide/etendo-classic/basic-features/general-setup/application/region.png)

La solapa Provincia permite al usuario visualizar y/o crear y mantener las provincias de cualquier tipo de país.

!!! info
    Las provincias del país se definen simplemente mediante un nombre y una descripción.
