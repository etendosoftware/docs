# Rangos de conversión

:material-menu: `Aplicación` > `Configuración General` > `Aplicación` > `Rangos de conversión`


## Visión general

El rango de conversión, también llamado *tipo de cambio de moneda*, es el tipo al que una moneda puede convertirse en otra.

!!! info
    Los rangos de conversión solo pueden crearse a **nivel de organización del sistema (\*)**, lo que implica que están disponibles para todas las organizaciones dentro del cliente.

## Rangos

La ventana Rangos de conversión permite al usuario crear los rangos que se utilizarán para transacciones multimoneda.

Etendo requiere que el usuario introduzca y mantenga el tipo de cambio de moneda en ambos sentidos; esto significa introducir al mismo tiempo los dos rangos siguientes:

-  **Multiplicar por**: es el rango por el cual se multiplicará el importe base para calcular el importe convertido.
- **Dividir por**: es el rango por el cual se dividirá el importe base para calcular el importe convertido.

Por ejemplo, el tipo de cambio de `USD` a `EUR` debería tener una configuración de *Multiplicar por* = 0.68 y de *Dividir por* = 1.47, como se muestra en la imagen siguiente:
