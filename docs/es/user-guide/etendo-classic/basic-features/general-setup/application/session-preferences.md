# Preferencias Sesión

:material-menu: `Aplicación` > `Configuración General` > `Aplicación` > `Preferencias Sesión`

Las preferencias de sesión permiten mostrar u ocultar la Solapa contabilidad o las solapas de traducción para un **usuario**, **rol**, **entidad** y **organización** determinados.

![](../../../../../assets/drive/VAr7jhdNC8mm1JlSp-Jq-dBswsbSAMKpcuq_Tg6_Z3yKcxRbc3N8IgLn-gimlxHoPzGvtOl0i3k7wPCdqKKG8zO1xgcS5xvh-sC1T0yO2qeanxvBMqt-P8TUPDo9n-ASz4p2LjaL.png)

Tal y como se muestra en la imagen anterior, la ventana de preferencias de sesión permite al usuario configurar una lista de preferencias de sesión para el usuario, el rol, la entidad y la organización introducidos al iniciar sesión en Etendo.

Ese usuario deberá tener derechos de acceso a esta ventana para poder configurar estas preferencias.

Las preferencias de sesión disponibles que se pueden mostrar u ocultar son:

- **Habilitar contabilidad** (anteriormente denominada _Mostrar solapas de contabilidad_), esta opción permite mostrar las solapas definidas como *Solapa contabilidad* en una ventana; además, esta opción permite mostrar los botones de proceso que *Contabilizan* un documento.  

    La lista completa de solapas de contabilidad se muestra a continuación. Si se selecciona esta opción, se establece la preferencia **ShowAcct** en `Sí`.

    | Nombre de la ventana           | Nombre de solapa/subsolapa   |
    | ------------------------------ | ---------------------------- |
    | Esquema contable               | Esquema contable             |
    | Esquema contable               | Documentos                   |
    | Esquema contable               | General_Accounts             |
    | Esquema contable               | Valores por defecto          |
    | Datos de contabilidad          | Cabecera                     |
    | Categoría de Activos           | Contabilidad                 |
    | Activos                        | Contabilidad                 |
    | Terceros                       | Contabilidad de cliente      |
    | Terceros                       | Contabilidad de proveedor    |
    | Terceros                       | Contabilidad de empleado     |
    | Grupos de Terceros             | Contabilidad                 |
    | Cuenta financiera              | Historial contable           |
    | Cuenta financiera              | Configuración contable       |
    | Cuenta financiera              | Contabilidad                 |
    | Concepto contable              | Contabilidad                 |
    | Producto                       | Contabilidad                 |
    | Categoría del producto         | Contabilidad                 |
    | Rango impuesto                 | Contabilidad                 |
    | Almacén y ubicación            | Contabilidad                 |

- **Mostrar solapas de traducción**: esta opción permite mostrar las solapas de traducción. Las solapas de traducción se muestran en muchas ventanas de Etendo. Esta opción, si se selecciona, establece la preferencia "ShowTrl" en "Sí".
- **Mostrar auditoría**: del mismo modo, esta opción permite mostrar la información del rastro de auditoría. La información del rastro de auditoría también se puede encontrar en muchas ventanas de Etendo bajo la sección de grupo "Auditoría". Esta opción, si se selecciona, establece la preferencia "ShowAuditDefault" en "Sí".
- **Tema o skin**: existen dos opciones posibles, el skin "Valor por defecto" o "Classic" o "Compatible 2.50 a 3.00".
- **Rango de transacciones**: se refiere al número de días (contando hacia atrás desde el día actual) utilizado en el filtro implícito de pedidos de venta o pedidos de compra por ejemplo. Los filtros implícitos son los filtros utilizados para no mostrar documentos que se han completado en los últimos x días, para evitar desbordamientos.
- **"Rango de registros"** y **"Rango de registro del selector"** solo aplican a vistas 2.50. Etendo 3.0 permite desplazarse hacia abajo para ver todos los registros en una ventana, solapa o selector.

Por último, el botón **Guardar preferencias** permite guardar los cambios.
