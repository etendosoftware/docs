## Visión general

Este documento explica cómo IntelliJ IDEA es capaz de importar la configuración del formateador de código.

## Formato de código

1. En primer lugar, descargue el siguiente [archivo de formato de IntelliJ](../../../../assets/developer-guide/etendo-classic/getting-started/instalation/code-formatting-style-in-intellij/etendo.xml){: download="etendo.xml"}.
2. Abra :simple-intellijidea: Ajustes de IntelliJ IDEA
3. Busque Estilo de código → Java
4. Haga clic en el icono del engranaje → Importar esquema → XML de estilo de código de IntelliJ IDEA.

    ![estilo-de-código-java](../../../../assets/developer-guide/etendo-classic/getting-started/instalation/code-formatting-style-in-intellij/code-style-java.png)

5. Seleccione el archivo descargado primero e importe el esquema.
