    Creacin de transacciones desde otras transacciones

    Lo mismo sucede para las facturas. En Etendo se pueden crear facturas desde albaranes o pedidos. En este caso, el sistema siempre intentar recuperar la informacin de Intrastat del albarn o del pedido. Si la factura se crea sin albarn o pedido asociado, Etendo recuperar la configuracin del sistema para rellenar la solapa del Intrastat.

    Al procesar la declaracin, aparece automticamente el botn de Generar fichero.

    Cuando lo pulsemos, el sistema comprobar si estamos obligados a presentar la declaracin para este periodo o no. Para llevar a cabo esta operacin, el sistema comprueba el importe declarado el ao anterior y el acumulado de este ao, por ese motivo es muy importante que configure correctamente estos valores en la ventana Intrastat a nivel de Organizacin.

    Si durante el ao anterior superamos el lmite de declaracin de Intrastat, o lo hemos superado a lo largo del presente ao, Etendo nos generar el fichero de Intrastat con el formato apropiado para ser enviado telemticamente (a travs de Internet) a las Autoridades. Ejemplo:

    FR;31;FOB;11;3;;85182190;CN;1;115;162;15,37;15,37

    DE;28;CIF;11;1;0811;85182190;US;1;2459;1982;4589,46;4589,46

    IT;12;FOB;11;3;;02012030;;1;800;;987,00;890,45

    Info

    Si las lneas de la declaracin sobre la que vamos a generar el fichero son superiores a 1000, generar un fichero zip con tantos txt como sean necesarios, ya que cada archivo podr contener un mximo de 1000 lneas.

    Info

    Este fichero se puede importar en el enlace web de la AEAT - Cumplimentacin en Lnea con Importacin de Fichero. Para ms informacin, consulte este enlace.

    Info

    Es importante recalcar que superado el umbral de presentacin en caso de que en un periodo (mes) no se realicen transacciones de intercambio de bienes intracomunitarias, habr que presentar la correspondiente declaracin sin transacciones.

    Finalizar el ao. Copia de configuracin al siguiente ao

    Una vez que hemos generado todas las declaraciones del ao, es momento de configurar el sistema para el prximo ao. Lo nico que debe hacer es ir a la configuracin del presente ao en Configuracin General | Organizacin | Organizacin | Organizacin] >> Intrastat y presionar el botn Copiar al ao siguiente. En la ventana emergente aparecer un campo numrico para que introduzca el nuevo importe lmite del siguiente ao.

    El proceso comprobar que haya procesado todas las declaraciones del presente ao y, en ese caso, crear un nuevo registro copiando toda la informacin de configuracin del presente ao junto al nuevo importe lmite de declaracin de Intrastat.
