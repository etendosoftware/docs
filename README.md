# Etendo Documentation

This repository contains the source files for the official Etendo documentation, built with MkDocs.

# Requirements

Python 3.11

# First Install

```bash
python3 -m venv venv
source venv/bin/activate
pip install -r requirements.txt

```


# Run Etendo Documentation locally

```bash
source venv/bin/activate
mkdocs serve
```
