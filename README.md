# Etendo Documentation

This repository contains the source code and content for the Etendo documentation, built using MkDocs.

# Requirements

Python 3.11

# First Install

```bash
python3 -m venv venv
source venv/bin/activate
pip install -r requirements.txt

```


# Run Etendo Documentation locally

```bash
source venv/bin/activate
mkdocs serve
```
